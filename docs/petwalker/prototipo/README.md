# Protótipo navegável do MVP PetWalker

## Como visualizar
1. Abra o arquivo `docs/petwalker/prototipo/index.html` no seu navegador (clique duas vezes ou use uma extensão Live Server).
2. Use as abas superiores para alternar entre os perfis (Tutor, Passeador, Admin).
3. Selecione a tela desejada no carrossel de abas secundárias.
4. Explore os elementos interativos: botões, formulários, badges e mapas simulados.

## Dicas para apresentação
- Inicie pelo perfil Tutor para contar a jornada completa (onboarding → solicitação → tracking → avaliação).
- Em seguida mostre a perspectiva do Passeador para reforçar previsibilidade de ganhos.
- Finalize com o painel Admin destacando governança e segurança.
- Use os botões "One-pager" e "Inventário de telas" no topo do protótipo para abrir os materiais de apoio na mesma pasta.

## Limitações do protótipo
- Todo o conteúdo é estático; não há integração com APIs, marketplace ou IA.
- Informações de mapas e tracking são mockadas com placeholders visuais.
- Não há persistência de dados; cada recarregamento retorna ao estado inicial.
